一：游戏概要：
角色是固定的11人且所有人身份都已知，其中有两个玩家被赋予狼人身份和狼刀。
*第一个白天狼人不知道队友是谁，有可能自相残杀。
*早上各自选择地点行动，夜晚进聊天室。进聊天室前狼人有私下沟通的机会，聊天室聊天时按大小姐选择的顺序发言，每人120s，同时允许给一位玩家发一次文本私信。
*投票走的玩家有遗言，其他没有。

二：角色技能
1.大小姐：绝对不会是狼，聊天室阶段决定发言顺序，最后一个发言。死后虽无视角，且无法发言，但仍可以继续决定发言顺序。一局有一次激光，发射后聊天和投票流程不断，只可以在聊天室投票前杀人，且不可杀护。
2.护：每天可以守护一个人绝对不会死亡（投票除外）且视野不被遮蔽（防了一手妖精），不能连续两天守同一个人。
3.魔女：白天可以消耗行动点使用一次光波，发射光波会公示。若杀到队友则自己死亡。
4.侦探：白天等其他人选择完地点后再选，也就拥有了每个地点的人数信息。选后可以消耗行动点躲入墙内。若前往地点已满人则自动消耗行动点躲入墙内。在墙内能看到在场其他人，但无法行动或发动狼刀。另外，狼可以选择杀墙。
5.小原原：每天投票前选定一个玩家，使其投票无效，自己投票x2。一局2次。
6.蓝金：被动技能，若选择地点存在狼人，立刻警觉。每局第一次警觉时，免疫狼刀并打印当地所有玩家。
7.白雪：进聊天室前可以查验一个玩家是否为狼。
8.仓鼠：可以自爆带走周围一人（白天行动或聊天时均可），自爆后会公示。
9.人偶师：白天可以消耗一行动点复活一位死者，当晚死而复生者回归聊天室。一局一次。
10.厨娘：开局选择一位玩家绑定，绑定者不知道。若厨娘为好人，则与绑定者选择同一地点时可以免疫狼刀；若厨娘为狼人，则绑定者存活时免疫激光。
11.妖精：可以遮蔽一个人的视线，让其失去当日动刀机会和行动点。一局2次。

三：四个地点，每个地点最多三个人前往：
龙穴：可消耗行动点获取一颗龙心
厨房：可消耗行动点转让一颗龙心
商店：可消耗行动点消费2颗龙心验一个玩家是否为狼
图书馆：可消耗行动点消费1颗龙心免疫此次狼刀
注：若玩家被杀，持有龙心丢失；若被投，打印掉落龙心数量。 

四：游戏流程：
1.厨娘选择绑定者
2.每日顺序（角色名代表其发动技能）：
护，妖精，选择地点，侦探，各自知道在场人员，狼人动刀（不消耗行动点），所有人行动，报告死亡情况，白雪，狼人私聊，进聊天室，聊天or私信，小原原，投票

五：结局：
1.狼全灭，好人赢。
2.好人只剩一个除小原原外的人，或只剩大小姐激光用尽、人偶师技能用完的情况，都为狼人赢。小原原则有机会赢两狼。
3.小原原孤狼+至少三好人，若投票时发动技能后仍平票，判定为小原原被投出去。
4.仓鼠自爆导致无人生还，特别结局：无人生还。
